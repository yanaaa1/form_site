О проекте

Этот проект реализует интерактивную фотогалерею с управлением состоянием на стороне клиента (client-side state management).

  • Фильтрация по категориям и тегам.
  • Поиск по названию снимка.
  • Состояние фильтров кодируется в URL (query-параметры).
  • При обновлении страницы фильтры восстанавливаются из URL.
  • Семантическая вёрстка и адаптивный дизайн.

Стек: HTML + CSS + Vanilla JavaScript. Архитектура JS разделена логически на данные, состояние, рендеринг и обработчики событий, что упрощает расширение проекта и дальнейшую поддержку.

Идеи для расширения

  • Подгрузка фотографий с сервера (через fetch).
  • Модальное окно с большим фото и описанием.
  • Сортировка по дате или популярности.
  • Сохранение избранных фото в LocalStorage.
